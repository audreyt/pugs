Why is Pugs important for Perl 6?

  • The finaly implemenation of Perl 6 will be written in Perl 6. This compiler will be translated using pugs (bootstrapping)
  • The development of pugs helps to clarify and improve Perl 6's specification
  • A test suite (more than 18,000 tests at the time of writing, March 2007) is devloped along with pugs that will be Perl 6's offical test suite
