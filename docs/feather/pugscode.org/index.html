What is Pugs?

Pugs is an implementation of Perl 6, written in Haskell. It aims to implement the full Perl6 specification, as detailed in the Synopses. For more information, please see the overview and this interview.

Where Are We?

The Pugs project was started on February 1st, 2005 by Audrey Tang. To keep up to date on the latest developments, join us on IRC (irc.freenode.net #perl6), and check out This week on Perl 6.

How to get the code

Periodic releases will appear on CPAN under the Perl6-Pugs namespace.

For the very latest version of Pugs, check out the source from Subversion master repository, its mirror, or darcs. There is anonymous access to all. Commit access is handed out liberally; contact the Pugs team for details. (By the way, if you'd like offline working with the Subversion repository, the svk client may be of interest. But using vanilla svn is fine.)
