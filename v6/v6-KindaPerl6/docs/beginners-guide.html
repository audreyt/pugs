What do you need

  1. Perl5
  2. Coro.pm : http://search.cpan.org/~mlehmann/Coro-4.13/Coro.pm Required for t/kp6/internals/02-gather.t
  3. Subversion client.
  4. Commit bit: Ask on #perl6 irc.freenode.net.

Customs and Traditions:

  • (new comers) When you get your commit bit, Update the AUTHORS file to make sure everything (SVN) is ok.
  • Commit code in doubt, revert if wrong.
    http://irclog.perlgeek.de/perl6/2007-10-31#i_135203
  • Please prefix your kp6 svn commits with [kp6]
    http://irclog.perlgeek.de/perl6/2007-11-01#i_136077

    Example: 

    [kp6] I have refactored subroutine XXX for clarity and updated the documentation.

How to:

  • Download KindaPerl6. 
    svn co http://svn.pugscode.org/pugs/
  • To compile KP6. 
    perl Makefile.PL
make -j 3 test  -j 3 means do 3 jobs at once, great for systems with SMP
  • To re-compile KP6. 
    make clean
perl Makefile.PL
make -j 3 test
  • Force re-compilation of everything. 
    make forcerecompile
setenv PERL5LIB "./inc/v6-MiniPerl6-lib5" (XXX can this step be removed?)
make
    WARNING: Do NOT use -j when forcing a recompile!, this will cause the Makefile to crash. The make process will attempt to use code that has not yet been compiled.

  • To execute a script. 
    perl scripts/kp6 [perl6 file]
  • To compile a perl6 script and view resulting source: 
    perl scripts/kp6 -o [outputfile.pl] [source perl6 file]
  • To debug perl6 in perl5 code scripts. 
    setenv PERL5LIB compiled/perl5-kp6-mp6/lib
perl -d (some script)
DB> $DB::deep = 500;

Do to the recurisve nature of the code, you're likely to hit 100 pertty fast.

Cannot see the forest for the trees.

One of the problems I had when I started working on kp6 was I could not see how everything worked together.

mp6.pl (miniperl6) is used to compile/translate some of the KP6 perl6 code into perl5 code. The resulting perl5 code is used to provide a run time enviroment for code (a script or a one liner) translated by kp6 (script/kp6) into perl5 code.

Urk!. That translates into this.

  1. Compile kp6 (written in perl6) libraries (src/KindaPerl6/???) using miniperl6 to provide a run time foundation for code compiled by script/kp6.

    TODO: pmurias @tell dlocaus file in KindaPerl6::Runtime::Perl6 are compiled by kp6 (script/kp6) not mp6 IRC

  2. Use kp6 to compile src/KindaPerl6/Runtime/* These go into lib/compiled-kp6-mp6/KindaPerl6/Runtime/* ...

    These libraries provide an emulation of the perl6 enviroment.

    Miniperl6 converts only a small subset of perl6 into perl5 code.

  3. Compile a script script/kp6.

    perl script/kp6 -o perl5code.pl source-perl6.pl.

  4. Execute the resulting perl5code.pl using the lib/compiled-kp6-mp6/... libraries to emulate the perl6 enviroment, which is class creation, variable allocation, inheritance, etc...
