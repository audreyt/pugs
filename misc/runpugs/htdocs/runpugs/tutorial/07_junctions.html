Junctions

Allow you to test a variable against many conditions more easily.
The equivalent Perl 5 is 
unless $input eq 'y' | 'n' | 'c' {
    print "(y)es/(n)o/(c)ancel? ";
}
unless ($input eq 'y' ||
    $input eq 'n' ||
    $input eq 'c') {
    print "(y)es/(n)o)/(c)ancel? ";
}
Type :next to continue
