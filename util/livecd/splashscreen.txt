
=== What is Pugs?

Pugs is an implementation of Perl 6, written in Haskell. It aims to implement
the full Perl6 specification, as detailed in the Synopses, the design documents
of Perl 6. For more information, please see http://www.pugscode.org/.

=== What is Parrot?

Parrot is a virtual machine designed to execute bytecode for interpreted
languages efficiently. Parrot will be the target for the Perl 6 compiler.

=== What is on this cd?

This cd contains Pugs and Parrot. The system will boot directly into an
interactive Perl 6 shell.
