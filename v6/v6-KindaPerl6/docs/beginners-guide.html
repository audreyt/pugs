What do you need

  1. Perl5
  2. Coro.pm : http://search.cpan.org/~mlehmann/Coro-4.13/Coro.pm
    Required for t/kp6/internals/02-gather.t
  3. Subversion client.
  4. Commit bit: Ask on #perl6 irc.freenode.net.

Customs and Traditions:

  • (new comers) When you get your commit bit, Update the AUTHORS file to make sure everything (SVN) is ok.
  • Commit code in doubt, revert if wrong.
    http://irclog.perlgeek.de/perl6/2007-10-31#i_135203
  • Please prefix your kp6 svn commits with [kp6]
    http://irclog.perlgeek.de/perl6/2007-11-01#i_136077

How to:

  • Simply pull down the respository 
    svn co http://svn.pugscode.org/pugs/
  • To compile KP6 
    perl Makefile.PL
make -j 3 test
    -j 3 means do 3 jobs at once, great for systems with SMP

  • To re-compile KP6 
    make clean
perl Makefile.PL
make -j 3 test
  • Force re-compilation of everything code. 
    make forcerecompile
setenv PERL5LIB "./inc/v6-MiniPerl6-lib5"
make
    WARNING: Do NOT use -j when forcing a recompile!, this will cause the Makefile to break as code will be used that has not yet been compiled.

  • To execute a script 
    perl scripts/kp6 [perl6 file]
  • To debug perl6 in perl5 code scripts 
    setenv PERL5LIB compiled/perl5-kp6-mp6/lib
perl -d (some script)
DB> $DB::deep = 500;  # do the recurisve nature of the code, you're likely to
                      # hit 100 pertty fast.
